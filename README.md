# ApplyMe - Online Application Portal

A full-stack web app where users can submit applications and receive confirmation emails.

## Features
- Application form
- Store applications in SQLite database
- Email notifications

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQLite

## How to Run
1. Install Python and Flask
2. Run `python backend/app.py`
3. Open `frontend/index.html` in browser
4. Submit application and receive email
